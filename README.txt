In this project, I implemented basic tic tac toe, 9-board, and Ultimate tic-tac-toe. I'd like you to consider my implementation of ultimate TTT for extra credit since I am an undergraduate.



To compile, run:

bash compile.sh


Run directions: 

To run Basic Tic Tac Toe, type:

java Main

To run Advanced Tic Tac Toe, type:

java Main NineBoardTTT

To run Ultimate Tic Tac Toe, type:

java Main UltimateTTT
